Privacy in plain language

How the website and FamilyPD OS handle information.

Effective July 2026

At a glance

  • The public website is for learning, browsing, and practice.
  • Toolkit drafts use local browser storage.
  • FamilyPD OS uses Google sign-in for authentication.
  • FamilyPD does not sell household data or use it for advertising.
  • Do not store passwords or unnecessary sensitive information.

1. Public website

The public website provides Family PD information, pillar pages, resources, school-system guidance, youth learning options, project ideas, and browser-based toolkit demos.

2. Browser-based practice

Some public tools use local storage in the visitor’s browser. The information remains on that browser unless it is cleared, reset, or removed by browser settings. It is not the same as a permanent FamilyPD OS record.

3. FamilyPD OS authentication

FamilyPD OS may use Google sign-in to recognize the user opening the workspace. Google displays requested permissions before access is approved.

4. Google Drive features

When a user chooses a feature that creates or manages a FamilyPD file in Google Drive, the app needs permission for that user-requested action. FamilyPD OS does not need permission to read or send Gmail.

5. Data use

FamilyPD does not use household information for advertising, data sales, or unrelated profiling. Account data is used for authentication and to provide the workspace features requested by the user.

6. Children and families

Adults should review age requirements, privacy terms, public sharing, community features, and account settings before a child uses any linked platform. Do not enter passwords, Social Security numbers, full financial account details, or other unnecessary sensitive information into public demos or project forms.

7. External resources

Linked services have their own privacy, account, eligibility, pricing, and content rules. Review the official site before signing up or sharing information.

8. Contact

Questions may be sent to admin@familypd.org.
