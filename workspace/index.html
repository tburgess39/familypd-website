PDF workflow

Complete a tool once. Save an organized record.

1

Complete a tool

Enter meeting notes, pillar ratings, mission and vision, roles, or goals on the FamilyPD website.

2

Choose Generate PDF

The website sends only that completed tool’s information to the authorized Apps Script window.

3

Create two files

FamilyPD creates an editable Google Doc and a matching PDF in the user’s workspace.

4

Keep or remove

The user can open, organize, archive, or move FamilyPD-created files to Google Drive Trash.

Permission boundaries

Access is intentionally limited.

✓

Creates FamilyPD files

Folders, editable Google Docs, PDFs, and the workspace guide.

✓

Manages app-created items

Adds folders and moves FamilyPD-created files or folders to Trash.

✓

Keeps the user in control

Files remain in the user’s Drive and access can be revoked in the Google Account.

—

No Gmail permission

The workspace integration does not read, send, or manage email.

—

No arbitrary Drive cleanup

The manager is restricted to items created by the FamilyPD app.

Performance plan

Designed to avoid the lag we experienced in larger Apps Script builds.

One command at a timeA user-level lock prevents duplicate folders and double-click collisions.
Minimal initial buildOnly core folders are created at setup; deeper folders are created on demand.
Saved folder IDsThe app stores workspace IDs per user instead of searching all of Drive repeatedly.
Short-lived cachingStatus and managed-file lists are cached to reduce repeated Drive calls.
Single PDF requestEach export is handled in one server call instead of many small browser-to-server calls.
No background scanningThe app checks or repairs the workspace only when the user requests it.
