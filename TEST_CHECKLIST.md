# Test checklist

1. Deploy a new version and open the `/exec` app.
2. Open Project Studio and create/save a project.
3. Select **Connect Google Drive**.
4. If the workspace is healthy, confirm that a folder is created and **Open Google Drive folder** appears.
5. To test repair, trash or disconnect one app-created workspace item, then select **Connect Google Drive**.
6. Confirm FamilyPD shows the specific health issue and offers **Repair Workspace**.
7. Approve repair and confirm the app refreshes status and retries folder creation without reloading the browser.
8. Confirm project tasks and notes remain unchanged after any failed repair.
