# FamilyPD website connection update

1. Replace `/app/index.html` with the included file.
2. Replace `/resources/learning-tools/index.html` with the included file.
3. In `/app/index.html`, replace `REPLACE_WITH_FAMILYPD_OS_EXEC_URL` with the current Apps Script web-app `/exec` URL.
4. Commit and publish the website repository.
5. Test the public `/app/` page in a private browser window.

The `/dev` Apps Script URL is not suitable for the public website. Use the deployed `/exec` URL.
