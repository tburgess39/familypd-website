# What changed

- Preserved the existing FamilyPD OS features and services.
- Added Family Identity starter profiles so households do not begin with a blank page.
- Hid preference questions that did not yet change the experience, while keeping functional language and accessibility controls.
- Reframed Family Sharing in plain language with different Lead and Member experiences.
- Added a connected-household view and a warned disconnect-and-rejoin workflow for members.
- Changed the Resource Hub to dropdown-first browsing. Search is optional and all resources display without typing.
- Added Project Studio with age-appropriate idea generation, a project task tracker, optional on-demand Google Drive folder creation, completion, reflection, and an optional public review link.
- Updated the OS language to FamilyPD OS / Family Operating System and removed remaining “Coming Soon” wording from included website pages.
- Kept plain-language guidance, read-aloud infrastructure, mobile support, English/Spanish support, examples, and accessibility fields.
