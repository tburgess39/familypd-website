A simple way to use the toolkit

Choose. Discuss. Decide. Practice. Review.

1

Choose

Select one tool based on the current need.

2

Discuss

Make room for each person’s perspective.

3

Decide

Agree on one or two realistic actions.

4

Practice

Use the plan during everyday family life.

5

Review

Return to the tool and adjust what is not working.
