# FamilyPD Website/App Architecture Correction

## Main correction

The public website now focuses on:
- learning and exploration;
- public resources and examples;
- sample tools and practice;
- Kids & Teens project ideas;
- school and community connections.

The private FamilyPD App will handle:
- implementation;
- private family records;
- real meetings, goals, roles, policies, and progress;
- Google Drive workspace creation;
- Google Docs and PDFs.

Planned address: `app.familypd.org`

## Kids & Teens correction

Every project now:
1. starts with a household need;
2. identifies school-connected skills;
3. recommends official learning resources;
4. identifies school or community support;
5. includes family testing and improvement.

## Translation correction

The Creator Lab now translates:
- instructions;
- age bands;
- pillars;
- project types;
- time estimates;
- skill labels;
- household, school, and community connections;
- project steps;
- test questions;
- resource guidance;
- practice-plan labels.

## Learning resources added

The new directory includes Code.org, Khan Academy, Microsoft MakeCode, Scratch, MIT App Inventor, Canva for Education, OpenAI Academy and ChatGPT, Grok, Diffit, and MagicSchool.

## Public/private separation

Direct Google Drive controls and workspace scripts were removed from public pages.

See:
- `SITE_APP_ARCHITECTURE.md`
- `REMOVE_FROM_PUBLIC_REPO.md`
